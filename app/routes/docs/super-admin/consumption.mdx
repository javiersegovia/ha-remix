# Consumo

Por último entramos a la sección de consumo, en la que tendremos la visual de una lista contemplada por varios renglones obligatorios a rellenar para la carga del archivo .csv

![image from companies](/images/docs/consumption.png)

En la parte derecha de la página encontramos un botón amarillo con el nombre "Subir consumo" este desplegará un menú en el lateral derecho el cual permitirá hacer una carga masiva de estos consumos para así visualizarlos en la página principal de consumo.
