# Colaboradores

Acá en la segunda opción de actualización de la compañía encontramos un listado de los colaboradores vinculados a la misma, en esta sección podremos tanto crear, cargar, actualizar y visualizar a estos colaboladores.

## Crear colaboladores

Para la creación de un nuevo colaborador debe clickear en el botón amarillo " + Nuevo colaborador " que se encuentra arriba a la derecha del listado.
![image from companies](/images/docs/butom.png)

Se desplegará una nueva página donde estarán todos los campos a rellenar con la información del colaborador:

### Cuenta de usuario

- Nombre: colocar el nombre del colaborador que sale en su documento de identidad, este campo es obligatorio.
- Apellido: colocar el apellido del colaborador que sale en su documento de identidad, este campo es obligatorio.
- Correo electrónico: agregar el correo electrónico del colaborador con el cual ingresará a la plataforma, este campo es obligatorio.
- Contraseña: ingrese una contraseña para el acceso del colaborador al portal, la contraseña no tiene ninguna restricción queda de tu parte que tan complicada sea.
- Estado: existen dos opciones, activa o inactiva dependiendo de si el colaborador ya pertenece a la comunidad de HoyTrabajas escogerás la opción indicada.
- Fecha de ingreso: en este campo esta la posibilidad de buscar en el calendario la fecha exacta en la que el colaborador ingresará a la compañía.
- Fecha de retiro: en este campo esta la posibilidad de buscar en el calendario la fecha exacta en la que el colaborador se retirará a la compañía. De no saber esto información es mejor no rellenar el campo.
- Membresía: (campo que se cambia), este campo es obligatorio.

![image from companies](/images/docs/newEmployee.png)
Ejemplo visual de como rellenar la sección de cuenta de usuario

### Información complementaria

- Cargo: ingresar el cargo que ocupa el colaborador en la compañía, de no salir entre las opciones ir a la página de Data para crear este cargo.
- Área: ingresar el área que ocupa el colaborador en la compañía, de no salir entre las opciones ir a la página de Data para crear el área.
- País: en este campo se deberá colocar la nacionalidad del colaborador.
- Departamento: ingresar el departamento que ocupa el colaborador en la compañía, de no salir entre las opciones ir a la página de Data para crear el departamento.
- Ciudad: en este momento no disponemos de ciudades.
- Dirección: acá tendrás que colocar la dirección completa de residencia que tenga el colaborador actualmente.
- Número de celular: colocar el número teléfonico del colaborador incluyendo el código del país.
- Género: aquí seleccionas el género del colaborador.
- Fecha de nacimiento: en este campo es posible usar el calendario para ingresar facilmente la fecha en la que nació el colaborador.
- Fecha de expedición de documento: en este campo es posible usar el calendario para ingresar facilmente la fecha en la que se vence el documento de identidad del colaborador.
- Cantidad de hijos: ingresar la cantidad de hijos que posee el colaborador, en caso de no tener ninguno marcar 0 en la casilla.

![image from companies](/images/docs/compInfo.png)
Ejemplo visual de como rellenar la sección de información complementaria

### Cuenta bancaria

- Banco: colocar el nombre exacto del banco que usa el colaborador.
- Tipo de cuenta: escoger entre las opciones el tipo de cuenta que posee el colaborador.
- Número de cuenta: ingresar de manera precisa el número de cuenta que tiene el colaborador.
- Tipo de documento: escoger que documento específico desea usar el colaborador para identificarse.
- Documento de identidad: colocar el número de documento de identidad que se escogió en el campo anterior.

![image from companies](/images/docs/bank.png)
Ejemplo visual de como rellenar la sección de cuenta bancaria

### Salario en moneda fiat

- Salario: en este campo se coloca la cantidad exacta que tiene el colaborador como salario, por favor colocar la cantidad en pesos colombianos (COP).
- Moneda: acá escogerás entre las monedas disponibles (COP es la única por los momentos)
- Cupo aprobado en moneda fiat: cantidad de dinero aprobado para usar con los beneficios de adelantos de nómina y prima
- Cupo disponible en moneda fiat: cantidad de dinero disponible para usar con los beneficios de adelanto de nómina y prima

![image from companies](/images/docs/fiat.png)
Ejemplo visual de como rellenar la sección de salario en moneda fiat

## Cargar colaboradores

Existe otro botón aparte del de crear y es el de cargar colaboradores ![image from companies](/images/docs/loadEmployee.png)
Este permite que desde un archivo excel con un formato especifico (pedir formato) se haga una carga masiva la cual incluirá a todos los colaboradores dentro de la compañía que se este colocando.

## Eliminar colaborador

Esta opción a simple vista no podrá ser visualizada ya que al eliminar un colaborador se estaría borrando información sensible, así que esta está deshabilitada para el uso común, en caso de querer eliminar al colaborador tendrá que reportarselo a un administrador o alguien con la suficiente herarquía para aprobar o rechazar su petición.

## Actualizar colaborador

Esta sección de actualización será igual que la de crear, para ingresar a ella solo hace falta clickear sobre el colaborador que desee modificar e inmediatamente será redireccionado a la página con la información que ya habia sido suministrada para ese colaborador.
