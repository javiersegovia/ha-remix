# Subproductos

Para esta página observaremos los subproductos desplegados en una lista siendo posible interactuar con ellos en la creación, actualización y eliminación.
![image from companies](/images/docs/subproduct.png)
Ejemplo visual de la página de los subproductos.

## Crear subproducto

Tendremos un botón amarillo con el nombre de "+ crear subproducto" el cual abrirá un menú por el lateral derecho, este tendrá dos campos los cuales integrarán la información del subproducto.

- Nombre: campo el cual se rellenará con el nombre del subproducto a crear.
- Descuento: colocar el porcentaje de descuento que tendrá este subproducto.
  ![image from companies](/images/docs/createSubproduct.png)
  Ejemplo visual de como rellenar los campos de creación de subproducto.

## Actualizar y eliminar subproducto

Para la actualización nos encontraremos con un menú lateral similar al de la creación, con la variante de que habrá un botón el final del menú para eliminar el subproducto.

![image from companies](/images/docs/actSubproduct.png)
Ejemplo visual de como rellenar la sección de actualización del subproducto.
