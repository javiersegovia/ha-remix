# Configuración general

En esta página de configuración conseguimos 3 campos a ser rellenados según se haya acordado como debe de ser. Estos campos son:

- Tasa de interés anual: acá deberá ingresar el porcentaje del interes anual el cual ya fue acordado, de no saberlo contactar con un superior que tenga la información.
- Días sin solicitudes antes de una fecha de pago: escoger cuantos días de antelación hay antes de una fecha de pago para poder volver a recibir solicitudes nuevas.
- Auxilio de Transporte (Adelanto de Prima): suma acordada que se sumará al total del adelanto de prima como un interés.
