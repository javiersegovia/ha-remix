# Modelos de la base de datos

Visualización de una lista con todos los modelos de la base de datos existentes en la plataforma.

![image from companies](/images/docs/data.png)
Ejemplo visual de la página de data.

Para cada uno de los modelos habilitados al ser seleccionados entrarás a otra lista en la cual podrán ser creados, actualizados o borrados nuevos datos para la lista. Estos solo tendrán un campo el cual será el nombre, el cual se desplegará en un menú por el lateral derecho tanto en la creación como en la actualización y eliminación.
![image from companies](/images/docs/actData.png)
Ejemplo visual de la actualización de la información dentro de un modelo de base de datos.

## Categorías de beneficios

Este modelo en específico tendrá dos campos añadidos tanto en la sección de creación como de actualización, estos son:

- Color HEX: aquí tendrás que escoger un color para la ser expuesto en el dashboard de clientes en la categoría de beneficio correspondiente, este color tiene que ser escrito en su forma hexadecimal. Ejemplo: #0000ff color azul.
- Opacidad: escoger la transparencia que tendrá esta categoría de beneficios en el dashboard de clientes, escoger entre el 0 y el 1 esta opacidad.

![image from companies](/images/docs/category.png)
Ejemplo visual de la creación de la categoría de beneficios.
